Git Help

  • git config --global user.name blabla - команда для вводу імені користувача
  • git config --global user.email blabal@bla - команда для вводу пошти користувача
  • git init - ініціалізація у даному репозиторії
  • git sratus - перевірка вмісту локального сховища
  • git add - додавання файлів для подальших змін
  • git commit - збереження змін у сховищі
  • git log - інформація про зміни у поточній гілці
  • git checkout - перехід між гілками
  • git branch - список гілок
  • git push - відправка файлів до віддаленого репозиторія
  • git pull - стягування файлів з віддаленого репозиторія
  • git merge - команда для об'єднаня гілок
  • git clone - клонування віддалех даних до локальної папки
  • git diff - команда для перегляду змін
  • git remote - показує віддалені сховища
  • .gitignore - список файлів, які програма повинна проігнорувати
  • git revert - запис нових комітів, що скасовує деякі попередні
  • git reset - скасування комітів та збереження змін
  • git rebase - команда для зміни одонго коміта на інший
  • git cherry-pick - дозволяє вибрати коміт з однієї гілки і застосувати його на іншій
  • git stash - відкладає зміни, щоб застосувати їх пізніше
